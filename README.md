# 🧑‍💻 Employee CRUD App

A simple full-stack Employee Management web app using:

- AngularJS for frontend
- Node.js + Express.js for backend
- MongoDB for database

## 🚀 Features

- Add new employees
- View employee list

## 🛠 Technologies Used

- AngularJS
- Node.js
- Express.js
- MongoDB

## ▶️ How to Run

1. Start MongoDB server
2. Run backend:
   ```bash
   node server.js
